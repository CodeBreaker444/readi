A **System** in Readi is any physical asset tracked as a `tool` record in the database.

## System Fields

| Field | DB Column | Meaning |
|---|---|---|
| Asset code | `tool.tool_code` | Unique identifier (e.g. `DJI-M3T-001`) |
| Name | `tool.tool_name` | Display name |
| Active flag | `tool.tool_active` | `Y` = in service, `N` = decommissioned |
| Location | `tool.location` | Physical storage location |
| Activation date | `tool_metadata.activationDate` | When the asset entered service |
| Client assignment | `tool_metadata.clientId` | Which client this drone is dedicated to |
| Calibration | `tool.last_calibration_date` / `tool.next_calibration_date` | Calibration tracking |
| Warranty | `tool.warranty_expiry` | Warranty end date |

Each system can have **components** (`tool_component`): motors, batteries, sensors — each tracked individually for usage hours and expiry.

---

## Operational Status

Computed live on every fleet load. **Not a stored field.**

```
NOT_OPERATIONAL  >  MAINTENANCE  >  OPERATIONAL
```

| Status | Trigger condition |
|---|---|
| `OPERATIONAL` | No open tickets, no expired components |
| `MAINTENANCE` | Open maintenance ticket (`ticket_status != CLOSED`) OR component with `component_status = MAINTENANCE` |
| `NOT_OPERATIONAL` | Active component (`component_active = Y`) has `expiration_date <= today` |

<Warning>
A system can satisfy both MAINTENANCE and NOT_OPERATIONAL simultaneously. NOT_OPERATIONAL always wins the display.
</Warning>

See [System Status Rules](/platform-rules/system-status-rules) for the full computation.

## Decommissioned

**Manual:** Admin sets `tool_active = N` via Settings → Systems.

**Automated:** Not currently automated. The platform sends expiration notifications to managers when a component expires, but does not auto-decommission.

| State | `tool_active` | In fleet? | Missions? |
|---|---|---|---|
| In service | `Y` | Yes | Yes |
| Decommissioned | `N` | No | No |
| Soft-deleted | `Y` but `tool_metadata.deleted = true` | No | No |
| Warehouse | `Y` but `tool_metadata.is_warehouse = true` | No | No |

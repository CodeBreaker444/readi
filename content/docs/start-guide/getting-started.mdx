## To begin using Readi

Use this quick setup sequence to get operational:

1. Open **Dashboard** to review your operational overview
2. Set up **Company**, **Organisation**, and **Team** records
3. Configure **Systems** and **Components** before operations
4. Add **Training** courses and assign users where needed
5. Start mission planning from **Planning Dashboard**
6. Track execution from **Operations** boards, calendar, and logbooks
7. Maintain safety and audit readiness from **Compliance** modules

## Suggested first-week checklist

- Invite team members and verify access roles
- Complete baseline system and maintenance data entry
- Add at least one training event in the calendar
- Create one planning mission and walk it through operations
- Attach one evidence item in compliance to validate the workflow

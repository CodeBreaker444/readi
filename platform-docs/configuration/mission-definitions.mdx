---
title: "Mission Definitions"
description: "Configure mission statuses, types, categories, and result types."
---

Organisation-defined lookup values that power mission tracking.

## Configurable sets

| Set | Key field |
|---|---|
| **Mission Statuses** | `is_final_status`, `status_order` |
| **Mission Types** | `type_name` |
| **Mission Categories** | `category_name` |
| **Result Types** | `result_type_desc` |

Each set is scoped to the organisation.

## Primary app routes

- `/settings/mission-statuses`
- `/settings/mission-types`

## Main capabilities

- Create and manage user accounts (name, email, role, timezone)
- Role assignment: PIC, OPM, SM, AM, CMM, MM, TM, DC, SLA, ADMIN
- Viewer and Manager flags for additional permission layers
- Territorial unit assignment
- Resend activation invites to pending users

## User activation flow

New users receive an activation email with a one-time link. Until activated, they appear as **Pending**. Use "Resend Invite" if the email was missed.

## Primary app routes

- `/settings/team`

---
title: "FlytBase Integration"
description: "Real-time telemetry and ATC feed from FlytBase."
---

Readi integrates with FlytBase ATC for real-time drone telemetry during active missions.

## What it provides

- Live altitude, battery, and GPS feeds written to active `pilot_mission` records
- Incident flags set automatically: `link_loss`, `rth_unplanned`, `deviation_flag`
- Alert ingestion: FlytBase alerts (e.g. `WIND_ALERT`, `LINK_LOSS`) stored in `alert_log`

## Configuration

FlytBase credentials are configured at the organisation level via the FlytBase API key in organisation settings.

## Primary app routes

- `/integrations/flytbase`

Logbooks provide a searchable record of all completed missions.

## Main capabilities

- Filter by pilot, drone/system, mission type, category, result, status, and date range
- Each row shows: pilot, drone code, flight duration, distance, mission plan, client, status
- Drone status shown per entry — reflects the system's status at filter load time
- Export support for compliance and audit use

## Primary app routes

- `/logbooks/operation`

title: "Logbooks"
description: "Flight logbooks with multi-axis filtering."
---

Searchable record of all completed missions.

## Main capabilities

- Filter by pilot, drone, mission type, category, result, status, and date range
- Each row shows: pilot, drone code, flight duration, distance, mission plan, client, status
- Drone status reflects the real value from `tool_status.status_code`
- Export for compliance and audit use

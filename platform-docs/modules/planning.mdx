title: "Planning"
description: "Flight plan creation, approval workflow, and mission scheduling."
---

Planning is where flight operations begin. Plans must reach `POSITIVE_RESULT` before a live mission can be created.

## Main capabilities

- Create and manage flight plans with waypoints and flight parameters
- Approval workflow: `NEW → PROCESSING → REQ_FEEDBACK → POSITIVE_RESULT / NEGATIVE_RESULT`
- Test logbook for rehearsal flights before live operations
- Assign drones and payloads to plans via Mission Component

## Primary app routes

- `/planning`
- `/planning/logbook`

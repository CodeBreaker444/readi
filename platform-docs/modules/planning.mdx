Plans must reach `POSITIVE_RESULT` before a live mission can be created.

## Approval workflow

`NEW → PROCESSING → REQ_FEEDBACK → POSITIVE_RESULT / NEGATIVE_RESULT`

## Main capabilities

- Create flight plans with waypoints and flight parameters
- Test logbook for rehearsal flights before live operations
- Assign drones and payloads to plans via Mission Component

## Primary app routes

- `/planning`
- `/planning/logbook`

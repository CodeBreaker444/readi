---
title: "Audit Logs"
description: "Platform-wide event log for every create, update, and delete action."
---

## What is logged

- User created / updated / deleted
- Mission status changes (who changed it and when)
- Compliance status changes
- Document uploads
- System and component changes

## Fields per entry

| Field | Meaning |
|---|---|
| `event_type` | CREATE, UPDATE, DELETE |
| `entity_type` | What was changed |
| `entity_id` | ID of the changed record |
| `description` | Human-readable summary |
| `user_name` / `user_email` | Who performed the action |
| `created_at` | When it happened |

## Primary app routes

- `/records/audit-logs`

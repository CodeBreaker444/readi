Settings and Security covers access hardening and API key management.

## Main capabilities

- Security settings and password policy flows.
- API key management for integrations.
- MFA related setup and verification paths.

## Primary app routes

- `/settings/security`
- `/auth/setup-2fa`
- `/auth/verify-mfa`
- `/auth/change-password`

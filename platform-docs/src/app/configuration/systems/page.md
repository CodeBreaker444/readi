Systems and maintenance controls assets, component structure, and maintenance operations.

## Main capabilities

- System and component inventory management.
- System map for location/context visualization.
- Maintenance dashboard and ticket lifecycle workflows.

## Primary app routes

- `/systems/manage`
- `/systems/map`
- `/systems/maintenance-dashboard`
- `/systems/maintenance-tickets`

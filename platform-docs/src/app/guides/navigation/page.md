This guide helps you efficiently navigate Readi for common operational tasks.

## Main Navigation
The primary navigation menu is located on the left side of the interface.

- **Dashboard**: Operational and safety overview
- **Planning**: Evaluation, mission templates, flight requests, and planning missions
- **Operations**: Daily board, operations calendar, and operation table
- **Compliance**: Audit plan, requirement evidences, and safety target review
- **Training**: Courses and training calendar
- **Systems**: System map, component management, and maintenance
- **Organization/Team**: Personnel, shifts, assignments, and communication

## Common Tasks
## Operations and Planning

| If you want to...             | Go to...                            |
|------------------------------|--------------------------------------|
| See operational overview      | Dashboard                           |
| Create flight requests        | Planning > Flight Requests          |
| Build a mission plan          | Planning > Planning Mission         |
| Track current operations      | Operations > Daily Board            |
| Review mission history        | Logbooks > Operation Logbook        |

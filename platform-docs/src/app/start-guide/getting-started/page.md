## To begin using Readi

Use this quick setup sequence to start working in the platform:

- Open **Dashboard** to review your operational overview.
- Set up **Company**, **Organization**, and **Team** records.
- Configure **Systems** and **Components** before operations.
- Add **Training** courses and assign users where needed.
- Start mission planning from **Planning Dashboard**.
- Track execution from **Operations** boards, calendar, and logbooks.
- Maintain safety and audit readiness from **Compliance** modules.

## Suggested first-week checklist

- Invite team members and verify access roles.
- Complete baseline system and maintenance data entry.
- Add at least one training event in the calendar.
- Create one planning mission and walk it through operations.
- Attach one evidence item in compliance to validate workflow.

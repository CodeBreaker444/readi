## Core Feature Areas

- **Planning and Evaluation**: Build mission plans, run evaluations, manage flight requests, and coordinate pre-flight activities.
- **Operations Management**: Work with daily boards, operations calendars, operation tables, and mission/operation logbooks.
- **Safety and Compliance**: Track SPI/KPI definitions, review safety targets, manage audit plans, and attach evidence for compliance requirements.
- **Team and Organization**: Manage personnel, crew shifts, qualifications, assignments, communication, and organizational checklists.
- **Training Management**: Handle courses, attendance, user progress, training calendar events, and automatic recompute workflows.
- **Systems and Maintenance**: Maintain systems/components inventory, map view, maintenance dashboard, and ticket lifecycle management.
- **Document Repository**: Store and manage controlled documents with revision support and access tracking.
- **Audit and Traceability**: Use audit logs and event history for accountability and operational transparency.
- **Security and Access**: Support authentication flows including password updates and MFA setup/verification.

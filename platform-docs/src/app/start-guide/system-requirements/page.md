## Readi is a web-based platform that works best with

- Browser: Chrome 90+, Firefox 88+, Edge 90+, Safari 14+
- Hardware: Modern CPU with at least 8 GB RAM for smooth data-heavy tables and dashboards
- Internet: Stable broadband connection for real-time operations and document workflows
- Display: Minimum 1366x768 resolution; 1920x1080 recommended
- Security: Enable multi-factor authentication for privileged users

For best team performance, use updated browsers and avoid running Readi in outdated webview environments.

## Feature areas

| Area | What it covers |
|---|---|
| **Planning** | Flight plan creation, approval workflow, waypoints, test logbooks |
| **Operations** | Kanban board, daily calendar, mission execution tracking |
| **Logbooks** | Flight logbooks with filtering by pilot, drone, date, type |
| **Systems** | Drone and equipment inventory, component lifecycle, maintenance tickets |
| **Safety** | Incident reports, hazard observations, corrective actions, KPI tracking |
| **Compliance** | Audit records, regulatory requirements, evidence upload |
| **Training** | Course definitions, attendance, certification tracking |
| **AI Agent** | Role-scoped natural language queries over mission, maintenance, and compliance data |

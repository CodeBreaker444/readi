## Browser support

Readi is a web-based platform. Use a modern browser for the best experience:

| Browser | Minimum version |
|---|---|
| Chrome | 110+ |
| Firefox | 110+ |
| Edge | 110+ |
| Safari | 16+ |

## Network

- A stable internet connection is required for real-time telemetry (FlytBase integration)
- Supabase and Resend outbound connections must not be blocked by your firewall

## Roles and access

All features are role-gated. Ensure each team member is assigned the correct role before giving them access. See the [Role Glossary](/business-logic/role-glossary) for the full list.

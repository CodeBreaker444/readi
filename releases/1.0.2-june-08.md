## 🚀 What's New

### Non-Operational Systems — Visual Warnings Across the App
- Operation Calendar: missions linked to a non-operational system appear grayed out with a strikethrough
- Document Repository: documents shown grayed out with a "Not Operational" label; upload form shows a warning banner when the selected system is non-operational
- Mission Creation: non-operational drones are grayed out and cannot be selected; existing missions with a non-operational drone assigned show a clear warning banner

### Drone Classes — Now Independent Per Component
- Classes assigned to a component belong to that component only — removing from one does not affect others
- Global class manager is now a suggestions pool only and no longer controls individual component assignments
- Custom class values can be entered freely outside the global list

## What's New

### AI Agent — Decommissioned System Awareness

The AI agent can now answer questions about decommissioned systems and expired components.

Ask questions like:
- *"Which systems are decommissioned?"*
- *"Are there any expired components in my fleet?"*
- *"Which drones are no longer operational?"*

The agent will list every affected system along with the components that have expired, grouped clearly by drone. If everything is in order, it will confirm that no decommissioned systems were found.

---

### Operation Logbook — System Filter Now Shows Correct Status

Previously, all drones appeared as "Operational" in the logbook system filter regardless of their actual state. The filter now reflects each drone's real status, so non-operational or under-maintenance systems are correctly labelled when filtering logbook entries.

---

## Bug Fixes

### Logbook system filter showed wrong status for all drones

**Issue:** Every drone in the Operation Logbook filter appeared as "Operational" even when it was grounded for maintenance or had an expired component.

**Fix:** The filter now reads each drone's actual status from the platform, so what you see in the filter matches what is shown on the Fleet and Operations pages.

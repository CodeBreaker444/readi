## What's New

### A Cleaner Import Mission Wizard

The Import Mission wizard has been tidied up — an old "Recurring (Weekly)" option that no longer applied to imported missions has been removed, so the form only shows fields that are actually relevant to bringing in a past flight.

### Open/PDRA Selection When Importing from the Operations Table

The Import Mission wizard, when opened from the Operations table, now lets you choose the operation type — **Open** or **PDRA** — just like the regular New Mission form. Previously this choice wasn't available there, so imported missions always defaulted to Open.

### See Which Mission a Flight Is Already Linked To

In Control Center's flight list, a flight whose log has already been attached to a mission is now clearly tagged, showing the mission it belongs to at a glance — no more guessing or having to open each flight to check.

## Improvements & Fixes

### Fixed: "Update Maintenance" Wasn't Working for Imported or Attached Missions

Missions created through Import Mission, or created by attaching a Control Center flight to a new mission, weren't being marked as **Completed** correctly behind the scenes. This quietly broke the "Update Maintenance" button for those missions — it looked available but did nothing. Missions created this way are now correctly marked Completed everywhere, and updating maintenance, viewing the attached log, and editing post-flight details all work the same way they do for any other completed mission — right from the same screen.

### Fixed: Duplicate or Conflicting Flight Logs

A mission can only ever have one flight log attached to it. Uploading a new log or attaching another Control Center flight to a mission that already has one is now blocked with a clear message, both when editing a mission from the Operations table and when completing a mission from the board — preventing logs from silently overwriting or conflicting with each other.

### Fixed: Attached Flight Maps Failing to Load

Viewing the map for a flight log that's already attached to a mission (from the System Management screens) used to reach out to Control Center live every time — and would sometimes fail with an error if that flight was no longer available there. These maps now load from our own secure storage instead, so they open reliably and instantly, even if the original flight is no longer accessible on Control Center's side.

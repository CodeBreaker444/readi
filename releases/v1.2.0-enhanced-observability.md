## 🚀 What's New

### Real-time System Alerts
- Instant notifications for flight system events
- Configurable alert thresholds per organisation
- Email and in-app delivery for critical alerts

### Team Mentions
- Tag team members directly in mission log comments
- Automatic notifications sent to mentioned users
- Activity tracking for all mentions

### Custom Dashboards
- Create and save personalised dashboard views
- Pin SHI KPIs and analytics widgets
- Dashboard layouts persist across sessions

### Audit Log Filters
- Advanced search and filter across all audit events
- Filter by user, role, action type, and date range
- Exportable filtered results

## 📈 Improvements

- Dashboard analytics load 40% faster
- Reduced API latency on flight request endpoints
- Better search performance across missions and logbooks
- Improved permission inheritance for nested roles

## 🐛 Bug Fixes

- Fixed audit log export failures for large date ranges
- Resolved role sync issue when updating team assignments
- Fixed missing timestamps on flight log entries
- Corrected Safari login timeout issue

## 🔐 Security

- Improved access token validation on API endpoints
- Hardened authorisation checks for flight data access

## ⚠ Breaking Changes

None

- Updated flight request API examples
- Expanded audit log filter reference
